NavThemes FontAwesome let you use Beautiful Fontawesome Icons on your site.

== Description ==
NavThemes FontAwesome a simple Gutenberg block Plugin which provides you two blocks of FontAwesome icons. Inspired by Gutenbee, This Plugin Makes adding FontAwesome Icons to your site breeze.

**Blocks:**

* **FontAwesome Icons**: Simple but powerful FontAwesome Block Plugin which let you add 1200+ Free FontAwesome Icons with So much add color and border variations. 
* **Social Media**: This let you add Social Links to your site with beacutiful Font Awesome icons.


== Installation ==
From within WordPress' dashboard:

1. Go to Plugins -> Add New
2. Search for "NavThemes FontAwesome"
3. Click "Install"
4. Click "Activate"
5. New FontAwesome Blocks will appear!
